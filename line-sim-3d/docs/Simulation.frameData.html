Class: FrameData

Simulation.FrameData()

Class used as a data class to store the current frame statistics. This includes dt, fps, and time since start.

Constructor

new FrameData()

constructor for the FrameData class. The data is automatically populated on creation using timing data from the previous frame that is stored in the Simulation.lastFrameTime and Simulation.firstFrameTime variables.
