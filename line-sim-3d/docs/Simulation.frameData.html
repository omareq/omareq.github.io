Class: FrameData

Simulation.FrameData()

Class used as a data class to store the current frame statistics. This includes dt, fps, and time since start.

Constructor

new FrameData()

constructor for the FrameData class. The data is automatically populated on creation using timing data from the previous frame that is stored in the Simulation.lastFrameTime and Simulation.firstFrameTime variables. If dt is greater than 100ms (for example when switching windows context, or by changing tabs) then dt is set to 10ms. Additionally if dt is greater than 16ms it is reduced to 16ms to maintain a constant sensor refresh rate. If the refresh rate drops below 50Hz then the robot moves too quickly over the line and misses it. Due to these time manipulations the time since start is different to the sum of all dts. Time since start is always the true difference between the start time and the current frame time.
