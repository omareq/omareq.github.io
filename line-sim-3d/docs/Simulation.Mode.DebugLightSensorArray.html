Class: DebugLightSensorArray

Simulation.Mode.DebugLightSensorArray()

Class Simulation.Mode.DebugLightSensorArray is a simulation mode that tests the light sensor array functionality. This is done by moving the light sensor array to the mouse location and constantly rotating it to make sure that it maintains the correct orientation and spacing for each sensor.
